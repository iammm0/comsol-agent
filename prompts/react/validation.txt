## 任务
验证给定的建模计划是否合理和完整。

## 验证项
请检查以下方面：
1. 步骤顺序是否合理（几何 → 物理场 → 网格 → 研究 → 求解）
2. 每个步骤的参数是否完整
3. 步骤之间是否存在依赖关系问题
4. 是否有遗漏的必要步骤

## 输出格式
请以 JSON 格式返回验证结果，包含以下字段：

{{
    "valid": true/false,
    "errors": ["错误列表"],
    "warnings": ["警告列表"],
    "suggestions": ["改进建议列表"]
}}

## 验证标准
1. 几何建模必须在物理场和网格之前
2. 物理场设置必须在研究配置之前
3. 网格划分必须在求解之前
4. 所有必要参数必须提供

请仔细验证计划，返回详细的验证结果。
