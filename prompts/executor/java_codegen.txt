## COMSOL Java API 要点

### 基础
1. 导入：com.comsol.model.* 与 com.comsol.model.util.*
2. 创建模型：ModelUtil.create(modelName)
3. 创建组件：model.component().create("comp1")

### 几何（2D）
- model.component("comp1").geom().create("geom1", 2) 表示 2D
- Rectangle：.set("size", new double[]{{w, h}}), .set("pos", new double[]{{x, y}})
- Circle：.set("r", radius), .set("pos", new double[]{{x, y}})
- Ellipse：.set("a", a), .set("b", b), .set("pos", new double[]{{x, y}})

### 几何（3D）
- model.component("comp1").geom().create("geom1", 3) 表示 3D
- Block：.set("size", new double[]{{w, d, h}}), .set("pos", new double[]{{x, y, z}})
- Cylinder：.set("r", radius), .set("h", height), .set("pos", new double[]{{x, y, z}})
- Sphere：.set("r", radius), .set("pos", new double[]{{x, y, z}})
- Cone：.set("r", rBottom), .set("rtop", rTop), .set("h", height)
- Torus：.set("rmaj", rMajor), .set("rmin", rMinor)

### 布尔运算
- Union、Difference、Intersection：.set("input", new String[]{{...}}), .set("keep", boolean)

### 拉伸 / 旋转
- Extrude：.set("input", ...), .set("distance", value)
- Revolve：.set("input", ...), .set("angle1", degrees)

### 材料
- model.materials().create("mat1")
- model.materials("mat1").propertyGroup("Def").set("thermalconductivity", value)
- model.materials("mat1").selection().all() 或 .set(new int[]{{1, 2}})

### 物理场
- model.component("comp1").physics().create("ht0", "HeatTransfer", "geom1")
- model.physics("ht0").create("bc1", "Temperature")
- model.physics("ht0").feature("bc1").set("T0", 293.15)

### 网格
- model.component("comp1").mesh().create("mesh1")
- model.component("comp1").mesh("mesh1").create("size", "Size")
- model.component("comp1").mesh("mesh1").feature("size").set("hauto", 5)
- model.component("comp1").mesh("mesh1").run()

### 研究
- model.study().create("std1")
- model.study("std1").create("std", "Stationary")
- model.study("std1").run()

### 选择集（Selection）
- model.component("comp1").selection().create("sel1", "Explicit")
- model.component("comp1").selection("sel1").geom("geom1")
- model.component("comp1").selection("sel1").set(new int[]{1, 2}) 或 .all()

### 几何导入（Import）
- model.component("comp1").geom("geom1").create("imp1", "Import")
- model.component("comp1").geom("geom1").feature("imp1").set("filename", path)
- model.component("comp1").geom("geom1").run()

### 网格（标签约定 mesh1 / geom1）
- model.component("comp1").mesh().create("mesh1", "geom1")
- model.component("comp1").mesh("mesh1").create("size", "Size")
- model.component("comp1").mesh("mesh1").feature("size").set("hauto", 5)
- model.component("comp1").mesh("mesh1").run()

### 结果导出
- model.result().export().create("img1", "Image")
- model.result().export("img1").set("filename", path).set("plotgroup", "pg1").run()
- model.result().export().create("data1", "Data"); ... .set("data", "dset1").run()

5. 保存：model.save(filePath)
